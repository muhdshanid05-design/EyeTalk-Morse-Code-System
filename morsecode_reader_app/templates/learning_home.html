📘 EyeTalk Learning – Morse Code Tutorial

This tutorial helps you learn Morse code step by step using eye movements.

1️⃣ What is Morse Code?

Morse code is a communication system that represents letters and numbers using dots (.) and dashes (-).

In EyeTalk, Morse code is generated using eye blinks.

2️⃣ Dot and Dash

Dot ( . ) → Short eye blink
Dash ( - ) → Long eye blink

Short blink = less time, Long blink = more time

3️⃣ Eye Blink Mapping

  • 👁️ Short Blink → Dot (.)
  • 👁️ Long Blink → Dash (-)
  • ⏸️ Pause → Letter completed
  • 👁️👁️ Very long blink → Clear text
